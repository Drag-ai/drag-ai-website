## 🔄 Converting to Other Formats

While SVG is recommended for its scalability and quality, you can convert to PNG/JPG if needed:

### Online Converters (Free)
- [CloudConvert](https://cloudconvert.com/svg-to-png)
- [SVG2PNG](https://svgtopng.com/)
- [Convertio](https://convertio.co/svg-png/)

### Recommended Export Sizes
- **Favicon:** 32×32, 64×64, 128×128 px
- **Social Media:** 512×512, 1024×1024 px
- **Website:** 256×256, 400×400 px
- **Print:** 2048×2048, 4096×4096 px

### Using Design Software
1. Open SVG in Figma, Adobe Illustrator, or Inkscape
2. Select File → Export
3. Choose PNG format and desired resolution
4. Export with transparent background if needed

## 📋 Usage Guidelines

### ✅ DO:
- Use logos in their original proportions (square aspect ratio)
- Maintain clear space around the logo (minimum 20% of logo size)
- Use on backgrounds that provide good contrast
- Scale proportionally when resizing

### ❌ DON'T:
- Distort or stretch the logo
- Change the gradient colors
- Add shadows or effects (logos already optimized)
- Use on busy backgrounds that reduce visibility
- Rotate or skew the logo
