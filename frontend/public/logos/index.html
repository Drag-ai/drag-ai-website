📖 Usage Guide

Why SVG Format?

  • Infinite scalability: Looks crisp at any size, from favicon to billboard
  • Small file size: Typically 1-5 KB, perfect for web performance
  • Editable: Can be opened and modified in design tools like Figma, Illustrator, or Inkscape
  • CSS customizable: Colors and effects can be changed with code

Converting to PNG (If Needed)

While SVG is recommended, you can convert to PNG using:

  • Online: CloudConvert.com, SVG2PNG.com, or Convertio.co
  • Design tools: Open in Figma/Illustrator and export as PNG
  • Recommended PNG sizes: 512×512 (web), 1024×1024 (social media), 2048×2048 (print)

Best Practices

  • Web headers: Use transparent or white background version
  • Social media: Use transparent version, 1024×1024 PNG export
  • Business cards: Use white background or transparent, vector format
  • App icons: Use white or dark background depending on OS theme
  • Print materials: Always use Ultra HD version for best quality
